A Bash script is a plain text file containing a series of commands written for the in a file we can be executed to achieve a result it also allow users to automate tasks by writing commands that the computer can execute sequentially. 
#!/bin/bash  This tells the system to use the Bash interpreter.

Add the commands you want the script to run.

Save the file with a .sh extension, for example, myscript.sh.

Make the script executable by running: chmod +x myscript.sh
Run the script using:
./myscript.sh
